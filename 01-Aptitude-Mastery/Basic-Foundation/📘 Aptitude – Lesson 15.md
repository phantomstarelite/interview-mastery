# 🧠 PART 1: Basic Idea of Mixture

A **mixture problem** involves:

* Two or more items
* Different prices or concentrations
* Mixed to form a new item

---

### Example (Basic)

Milk costs ₹40/litre
Water costs ₹0/litre

If mixed, average cost becomes somewhere **between 0 and 40**

📌 Final value always lies between given values.

---

# 🧠 PART 2: Weighted Average Method

### Formula

```
Average price = (Total cost) / (Total quantity)
```

---

### Example

10 litres at ₹20
20 litres at ₹30

```
Total cost = (10×20) + (20×30)
           = 200 + 600
           = 800

Total quantity = 30

Average price = 800 / 30 = ₹26.67
```

---

# 🧠 PART 3: Alligation Method (INTERVIEW SHORTCUT)

This is the **fastest method** for two items.

---

### Diagram Format

```
   Higher Price
        \      |
         \     |
          \    |
        Average
          /    |
         /     |
        /      |
   Lower Price
```

---

### Formula Logic

```
Ratio = (Difference crosswise)
```

---

### Example

Mix ₹40/litre milk with ₹25/litre milk
Final average = ₹30/litre

```
40      25
   \    /
     30
   /    \
```

Cross difference:

```
40 − 30 = 10
30 − 25 = 5
```

Ratio:

```
10 : 5 = 2 : 1
```

✔ Mix in ratio **2 : 1**

---

# 🧠 PART 4: Repeated Replacement (INTERVIEW FAVORITE)

> A container has milk.
> Some quantity is removed and replaced with water.
> Process repeated.

### Formula

If:

* Total quantity = Q
* Replaced each time = r
* Repeated n times

```
Milk left = Q(1 − r/Q)^n
```

---

# 🧠 PART 5: Important Properties

* Average always lies between values
* Use **alligation for 2 values only**
* For 3+ values → use weighted average

---

# 🧠 PART 6: Common Mistakes ❌

* ❌ Subtracting wrong direction in alligation
* ❌ Mixing ratio and quantity
* ❌ Forgetting total parts
* ❌ Using alligation for unequal quantity questions
