# 🧠 PART 1: What is a Ratio?

### 🔹 Definition

A **ratio** compares two quantities of the same type.

```
Ratio = a : b
```

Example:

* 10 boys and 5 girls
  Ratio = `10 : 5 = 2 : 1`

📌 Ratio has **no unit**

---

# 🧠 PART 2: Simplifying Ratio

Always divide by **HCF**

Example:

```
24 : 36
HCF = 12
= 2 : 3
```

---

# 🧠 PART 3: What is Proportion?

When two ratios are equal:

```
a : b = c : d
```

This is written as:

```
a/b = c/d
```

Cross multiply:

```
a × d = b × c
```

---

### Example

```
3 : 4 = x : 8
```

```
3/4 = x/8
3 × 8 = 4x
x = 6
```

---

# 🧠 PART 4: Dividing a Quantity in a Given Ratio

### Example

Divide ₹600 in ratio **2 : 3**

Step 1: Total parts

```
2 + 3 = 5
```

Step 2: Value of one part

```
600 / 5 = 120
```

Step 3:

```
2 × 120 = 240
3 × 120 = 360
```

✔ Answer: ₹240 and ₹360

---

# 🧠 PART 5: Direct Variation & Inverse Variation

### 🔹 Direct Proportion

If A increases, B increases.

Example:
More workers → more output

---

### 🔹 Inverse Proportion

If A increases, B decreases.

Example:
More workers → fewer days

---

# 🧠 PART 6: Compound Ratio (Interview Favorite)

If:

```
a : b = 2 : 3
c : d = 4 : 5
```

Compound ratio:

```
(2 × 4) : (3 × 5)
= 8 : 15
```

# ✅ Lesson 14 – Solutions

## 🔹 Q1

Simplify **45 : 60**

Find HCF:

```
HCF of 45 and 60 = 15
```

Divide both by 15:

```
45 ÷ 15 = 3
60 ÷ 15 = 4
```

✅ **Answer: 3 : 4**

---

## 🔹 Q2

Divide ₹840 in ratio **3 : 4**

### Step 1: Total parts

```
3 + 4 = 7
```

### Step 2: Value of one part

```
840 ÷ 7 = 120
```

### Step 3: Final values

```
3 × 120 = 360
4 × 120 = 480
```

✅ **Answer: ₹360 and ₹480**

---

## 🔹 Q3

If:

```
a : b = 5 : 7
b : c = 14 : 9
```

Make **b common**.

Since:

```
7 → 14  (×2)
```

Multiply first ratio by 2:

```
a : b = 10 : 14
b : c = 14 : 9
```

Now combine:

```
a : b : c = 10 : 14 : 9
```

✅ **Answer: 10 : 14 : 9**

---

## 🔹 Q4 (Interview Level 🔥)

Present age ratio:

```
A : B = 4 : 5
```

Let:

```
A = 4x
B = 5x
```

After 5 years:

```
(4x + 5) : (5x + 5) = 5 : 6
```

Convert to fraction:

```
(4x + 5) / (5x + 5) = 5/6
```

Cross multiply:

```
6(4x + 5) = 5(5x + 5)
24x + 30 = 25x + 25
```

```
24x − 25x = 25 − 30
−x = −5
x = 5
```

### Present ages:

```
A = 4x = 20
B = 5x = 25
```

✅ **Answer:**

* A = **20 years**
* B = **25 years**
