## 🧠 PART 1: Core Idea (VERY IMPORTANT)

When a boat moves in a river:

* The river has its own speed (called **stream speed**)
* The boat has its own speed (called **still water speed**)

---

## 🧠 PART 2: Key Definitions

### 🔹 Still Water Speed (u)

Speed of boat in **calm water**

### 🔹 Stream Speed (v)

Speed of river current

---

## 🧠 PART 3: Two Important Cases

### 🔹 Downstream (Boat + Current)

Boat moves **with the stream**

```
Downstream Speed = u + v
```

---

### 🔹 Upstream (Boat − Current)

Boat moves **against the stream**

```
Upstream Speed = u − v
```

📌 Always remember:

* Downstream is **faster**
* Upstream is **slower**

---

## 🧠 PART 4: Basic Example

A boat’s speed in still water = **20 km/hr**
Stream speed = **5 km/hr**

```
Downstream = 20 + 5 = 25 km/hr
Upstream   = 20 − 5 = 15 km/hr
```

---

## 🧠 PART 5: Finding Boat & Stream Speeds (INTERVIEW FAVORITE)

If given:

* Downstream speed (D)
* Upstream speed (U)

Then:

```
Still water speed (u) = (D + U) / 2
Stream speed (v)      = (D − U) / 2
```

---

### Example

Downstream = **18 km/hr**
Upstream = **12 km/hr**

```
u = (18 + 12) / 2 = 15 km/hr
v = (18 − 12) / 2 = 3 km/hr
```

✔ Boat speed = 15 km/hr
✔ Stream speed = 3 km/hr

---

## 🧠 PART 6: Time Problems

Use same formula:

```
Time = Distance / Speed
```

But use:

* Downstream speed for downstream time
* Upstream speed for upstream time

---

### Example

Distance = 30 km
Downstream speed = 15 km/hr

```
Time = 30 / 15 = 2 hours
```

---

## 🧠 PART 7: Very Common Interview Pattern

> A boat goes downstream in 2 hours
> and returns upstream in 3 hours
> Distance is same
> Find ratio of boat speed and stream speed.

---

### Step 1: Let distance = L

```
Downstream speed = L / 2
Upstream speed   = L / 3
```

### Step 2: Use formula

```
u = (D + U) / 2
v = (D − U) / 2
```

After solving:

```
u : v = 5 : 1
```

📌 These ratio questions are very common.

---

## 🧠 PART 8: Common Mistakes (AVOID ❌)

* ❌ Mixing upstream and downstream speeds
* ❌ Forgetting to divide by 2 in formulas
* ❌ Using same speed both directions
* ❌ Forgetting unit consistency

# Solution

# ✅ Lesson 13 – Solutions

## 🔹 Q1

Boat speed in still water = **18 km/hr**
Stream speed = **4 km/hr**

Formulas:

```
Downstream = u + v
Upstream   = u − v
```

Calculation:

```
Downstream = 18 + 4 = 22 km/hr
Upstream   = 18 − 4 = 14 km/hr
```

✅ **Answer:**

* Downstream = **22 km/hr**
* Upstream = **14 km/hr**

---

## 🔹 Q2

Downstream speed = **16 km/hr**
Upstream speed = **10 km/hr**

Formulas:

```
u = (D + U) / 2
v = (D − U) / 2
```

Calculation:

```
u = (16 + 10) / 2 = 26 / 2 = 13 km/hr
v = (16 − 10) / 2 = 6 / 2 = 3 km/hr
```

✅ **Answer:**

* Boat speed (still water) = **13 km/hr**
* Stream speed = **3 km/hr**

---

## 🔹 Q3

Boat travels **30 km downstream in 2 hours**

Formula:

```
Speed = Distance / Time
```

Calculation:

```
Downstream speed = 30 / 2 = 15 km/hr
```

✅ **Answer:**
Downstream speed = **15 km/hr**

---

## 🔹 Q4 (Interview Level 🔥)

Boat goes **24 km downstream in 3 hours**
Returns **24 km upstream in 4 hours**

### Step 1: Find speeds

```
Downstream speed = 24 / 3 = 8 km/hr
Upstream speed   = 24 / 4 = 6 km/hr
```

### Step 2: Find u and v

```
u = (D + U) / 2 = (8 + 6) / 2 = 7 km/hr
v = (D − U) / 2 = (8 − 6) / 2 = 1 km/hr
```

✅ **Answer:**

* Boat speed in still water = **7 km/hr**
* Stream speed = **1 km/hr**
